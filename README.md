# Project Overview

The Hospital Resource and Workflow Management System is a Java-based application developed to manage and organize different hospital operations. The system is designed to help hospital staff handle patient records, doctor schedules, appointments, billing operations, and department management in a structured way.

The application provides a centralized workflow for managing hospital activities and maintaining records. Different modules of the system work together to support daily hospital tasks such as registering patients, assigning doctors, scheduling appointments, generating bills, and handling department information.

The project follows a multi-class architecture where different functionalities are separated into controllers, services, models, utilities, and database-related components. The system uses a console-based interface for interaction and simulates real-world hospital management operations.

# System Modules

## Patient Management Module

This module manages patient-related operations such as:

* Adding new patients
* Updating patient records
* Searching patient details
* Viewing patient history

---

## Doctor Management Module

This module handles doctor-related information including:

* Doctor registration
* Doctor schedules
* Department assignments
* Availability management

---

## Appointment Management Module

This module manages hospital appointments by:

* Scheduling appointments
* Cancelling appointments
* Rescheduling appointments
* Managing appointment status

---

## Billing Module

The billing module is responsible for:

* Generating patient bills
* Calculating fees and discounts
* Managing payment records
* Displaying billing summaries

---

## Department Management Module

This module manages hospital departments including:

* Department records
* Department staff handling
* Department budget information
